USER MANUAL

In this page you can find the procedure allowing to CREATE, DISPLAY the list of users, EDIT and DELETE a User.

1. USER LOGIN :

1 - To log in, go to www.fabmstic-loans.com :

2 - Click on LOGIN.

3 - Enter your Identfiant and the corresponding Password.

4 - Click on LOGIN to Connect.

2. DISPLAY THE LIST OF USERS :

1 - To be able to display the list of users, you must be an Administrator and must be logged in.

2 - Then click on the ADMIN tab located in the navigation bar to access the DJANGO Administration Area.

2 - Now click on the USERS table to see the list of Users.

3 - And the list of users is displayed, on this same page you can see the ADD USER button which allows you to add a new user.

3. ADD NEW USER :

1 - To be able to display the list of users, you must be an Administrator and must be logged in.

2 - Then click on the ADMIN tab located in the navigation bar to access the DJANGO Administration Area.

2 - Now click on the USERS table to see the list of Users.

3 - And the list of users is displayed, on this same page you can see the ADD USER button which allows you to add a new user.

4 - To create a new user click on the ADD USER button.

5 - Fill in all the fields present in the form and click on the SAVE button to complete the CREATION of the user.

4. EDIT USER INFORMATION :

IMPORTANT : this small section explains the procedure to follow to assign one or more status to a user.

1 - To be able to display the list of users, you must be an Administrator and must be logged in.

2 - Then click on the ADMIN tab located in the navigation bar to access the DJANGO Administration Area.

3 - Now click on the USERS table to see the list of Users.

4 - Click on the name of the user you want to edit and the edit page containing the information of the entered user will be displayed.

5 - Edit the field or fields concerned and click on the SAVE button to save the modifications made.

5. EDIT YOUR PROFILE :

IMPORTANT : This section allows each user to edit their profile so they can change their username, last name, first name and email. Each user can only modify their profile.

1 - To access this functionality, you must be logged in.

2 - Once you are connected click on your username visible in the navigation bar and your profile will be displayed.

3 - Click on the EDIT PROFILE button to access the edit page of your profile.

4 - Edit the field or fields concerned and click on EDIT PROFILE button to save your changes.

6. CHANGE YOUR PASSWORD :

IMPORTANT : This section allows each user to edit their profile so they can change their username, last name, first name and email. Each user can only modify their profile.

1 - To access this functionality, you must be logged in.

2 - Once you are connected click on your username visible in the navigation bar and your profile will be displayed.

3 - Click on the CHANGE PASSWORD button to access the change password page.

4 - The password change page has three fields to fill in : current password, new password, confirm password.

5 - complete all fields and click on CHANGE PASSWORD button to save your changes.

7. FORGOT YOUR PASSWORD :

1 - To log in, go to www.fabmstic-loans.com :

2 - Click on LOGIN.

3 - Enter your Identfiant and the corresponding Password.

4 - If you forget your password, click on the RESET PASSWORD button.

5 - You will be redirected to a new page, enter the email address for your account and click on the RESET MY PASSWORD button to receive the link allowing you to create a new password.

6 - Fill in the fields : new password, confirm password click on the CHANGE MY PASSWORD button.
