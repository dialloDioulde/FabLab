USER MANUAL

In this page you can find the procedure allowing to CREATE, DISPLAY the list of Types, EDIT and DELETE a type.

All the operations that can be carried out on types are accessible to all users with an account.

1. CREATE A TYPE :

1 - To access this functionality, you must be logged in.

2 - Click on the DASHBOARD tab located in the navigation bar.

3 - You will be redirected to the page containing the list of all the Loans already made on the website.

4 - Move the mouse cursor completely to the left and click on the TYPE tab to access the list of all registered types.

5 - You also have access to the functionalities allowing to CREATE a Type, to EDIT the information of a Type and to DELETE a Type.

6 - To create a Type click on the ADD TYPE button, Fill in the fields : LAST MATERIAL TYPE, NAME TYPE, DESCRIPTION and click on the ADD TYPE button to complete the creation of the new type.

2. SHOW LIST OF TYPES :

1 - To access this functionality, you must be logged in.

2 - Click on the DASHBOARD tab located in the navigation bar.

3 - You will be redirected to the page containing the list of all the Loans already made on the website.

4 - Move the mouse cursor completely to the left and click on the TYPE tab to access the list of all registered types.

5 - You also have access to the functionalities allowing to CREATE a Type, to EDIT the information of a Type and to DELETE a Type.

3. EDIT A TYPE :

1 - To access this functionality, you must be logged in.

2 - Click on the DASHBOARD tab located in the navigation bar.

3 - You will be redirected to the page containing the list of all the Loans already made on the website.

4 - Move the mouse cursor completely to the left and click on the TYPE tab to access the list of all registered types.

5 - You also have access to the functionalities allowing to CREATE a Type, to EDIT the information of a Type and to DELETE a Type.

6 - Click on the EDIT button the type to edit to display its edit form with this information previously entered.

7 - Edit the field (s) concerned by the edition and click on the Save Changes button to save your modifications.

4. DELETE A TYPE :

1 - To access this functionality, you must be logged in.

2 - Click on the DASHBOARD tab located in the navigation bar.

3 - You will be redirected to the page containing the list of all the Loans already made on the website.

4 - Move the mouse cursor completely to the left and click on the TYPE tab to access the list of all registered types.

5 - You also have access to the functionalities allowing to CREATE a type, to EDIT the information of a type and to DELETE a type.

6 - Click the DELETE type button to delete.

7 - You will be asked to confirm the deletion, click OK to complete the action or click CANCEL to cancel the deletion.
